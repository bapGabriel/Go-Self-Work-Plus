# Go Self Work Plus

**Go Self Work Plus (GSWP)** é uma plataforma web que conecta clientes a prestadores de serviços autônomos, permitindo divulgação de serviços, comunicação direta, gestão de solicitações e avaliações.
Seu objetivo é facilitar a captação de demanda e gerar oportunidades para trabalhadores autônomos de forma simples e eficiente.

Projeto desenvolvido para a disciplina de Engenharia de Software.

## Requisitos

- Node.js 20+
- MongoDB

## Instalação

#### 1. Clone o repositório

#### 2. Configure as variáveis de ambiente:

Dentro das pastas back/ e front/, crie os arquivos .env copiando os exemplos (.env.example).

#### 3. Instale todas as dependências

A partir da pasta principal, rode:

```
npm run install:all
```

#### 4. Execute os projetos

Suba o backend e o frontend simultaneamente:

```
npm run dev
```
